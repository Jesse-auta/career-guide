# AI-Powered Career Advisor for IT Students

An intelligent chatbot built with **Google Gemini AI** and **Flask**, designed to help students navigate their career paths in Information Technology (IT). This assistant engages users in step-by-step, personalized conversations to recommend IT specializations, resources, and learning paths.

---

##  Features

-  Gemini-powered AI chat
-  Career-guidance questions asked one at a time
-  Tailored for beginners and students in IT
-  Flask backend API with frontend integration
-  CORS-enabled for cross-platform requests

---

## How It Works

1. The chatbot asks students targeted career questions.
2. Based on their answers, it suggests areas like:
   - Software Development
   - Cybersecurity
   - UI/UX Design
   - Data Science
   - Cloud Computing
3. It recommends beginner-friendly resources and tips.
4. The chat continues one question at a time until a recommendation is made.
